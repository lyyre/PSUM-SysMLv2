# Uncertainty Modeling for SysML v2  

This repository provides supporting materials for the paper titled *Uncertainty Modeling for SysML v2*.

- The [profile](profile) directory contains the MOF XMI specification of the PSUM–SysML v2 abstract syntax.
  > **Note:** `KerML_only_xmi.uml` and `SysML_only_xmi.uml` are sourced from the official SysML v2 Pilot Implementation repository:  
  > https://github.com/Systems-Modeling/SysML-v2-Pilot-Implementation
- The [case-studies](case-studies) directory includes the raw results of the case studies, including the original SysML v2 models and the corresponding models extended with PSUM–SysML v2.
